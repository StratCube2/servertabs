# 📑 ServerTabs (TMFS)

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Loader](https://img.shields.io/badge/loader-Fabric-green)
![Platform](https://img.shields.io/badge/platform-Minecraft-orange)

**Too many flippin' servers? Organize them in tabs.**

`ServerTabs` is a lightweight, customizable client-side mod for Minecraft (Fabric) designed to declutter your multiplayer menu. Instead of scrolling through a massive, endless list of servers, you can categorize them into organized, easy-to-access tabs.

---

## ✨ Features

* **🚀 Tabbed Organization:** Group your servers (e.g., Anarchy, SMP, Minigames, Friend Servers) into dedicated tabs.
* **🛠️ Highly Customizable:** Tailor your tabs and settings to fit your playstyle.
* **⚡ Quick Actions:** Move servers between tabs instantly using keyboard shortcuts.
* **🖥️ Clean UI:** A streamlined multiplayer menu that keeps your screen tidy.

## 🎮 How to Use

### Navigation
* **Cycle Tabs:** While in the Main Menu, hold `ALT` + `W` or `S` to quickly cycle through your created tabs.

### Managing Servers
* **Quick Assign:** In the tab dropdown menu, hold `ALT` and **click** on a specific tab. This will instantly move all servers from your "All" list into that chosen tab.

## 🛠️ Installation

1. **Requirements:** Ensure you have the [Fabric Loader](https://fabricmc.net/) installed for your version of Minecraft.
2. **Download:** Download the latest `.jar` file from the [Releases](https://github.com/StratCube2/servertabs/releases) page.
3. **Install:** Place the downloaded file into your `.minecraft/mods` folder.
4. **Launch:** Run Minecraft using the Fabric profile.

## ⚙️ Configuration

The mod includes a dedicated settings menu accessible via the Multiplayer screen. Here you can:
* Create new tabs.
* Rename or delete existing tabs.
* Customize visual preferences.
